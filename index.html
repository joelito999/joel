Documentation This is the help documentation. To view simply open index.html found inside this folder

Install Theme

Once you have downloaded the Circle package from Themeforest.net, there are 2 ways of installing this theme. These are as follows:

Method 1

  1. Unzip the download package using Winzip or equivalent.
  2. You will see another zip archive file inside called Circle.theme.zip, unzip this file also using Winzip or equivalent software and using an FTP client (eg. Filezilla), upload its contents to your WordPress installation. You will need to place the folder into the following location: wp-content/themes
  3. Now inside your WordPress admin area, go to Appearance > Themes and there you will see the Circle Theme.
  4. Click Activate to activate the theme. If you do not see the theme in this screen, you have done something incorrectly. Review the last few steps again.

Method 2

  1. Unzip the Circle download package. Once unzipped you will see a zip file named Circle.theme.zip, this is a compressed version of the theme.
  2. Now inside your WordPresss admin area, go to Appearance > Themes > Add new > Upload theme
  3. Click browse and locate this Circle.theme.zip file and upload it
  4. Now you will see the Circle theme in the Themes page.
  5. Click Activate to activate the theme. If you do not see the theme in this screen, you have done something incorrectly. Review the last few steps again.

NOTE: After install theme, you will need to install some plugins, just click Begin installing required plugins and active it.
